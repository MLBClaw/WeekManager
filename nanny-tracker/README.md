# 👩‍⚕️ 阿姨考勤助手

一款专为记录育儿阿姨考勤设计的移动端 Web 应用，支持华为手机及其他移动设备使用。

## ✨ 功能特性

### 📅 考勤记录
- **上班打卡**：一键记录阿姨上门工作
- **请假记录**：支持手动添加或语音记录请假
- **日历视图**：直观展示每月工作和请假情况

### 🎤 语音输入
- 支持语音记录请假（需浏览器支持 Web Speech API）
- 智能识别日期（今天、明天、昨天）
- 自动提取请假原因

### 📊 数据统计
- **工作天数**：自动统计本月工作天数
- **请假天数**：自动统计本月请假天数
- **预计工资**：根据日薪自动计算应发工资

### ⚙️ 设置管理
- 自定义日薪金额
- 设置每日工作时长
- 数据导出/导入（JSON 格式）
- 数据备份与恢复

## 🚀 使用方法

### 方式一：直接浏览器使用
1. 在手机浏览器中打开应用地址
2. 即可开始使用

### 方式二：添加到主屏幕（推荐）
**华为手机/安卓：**
1. 用浏览器打开应用
2. 点击菜单 → "添加到主屏幕"
3. 即可像原生 App 一样使用

**iPhone：**
1. 用 Safari 打开应用
2. 点击分享按钮 → "添加到主屏幕"

## 📱 应用截图

应用包含三个主要页面：

1. **首页**：打卡、日历、统计
2. **记录页**：历史记录列表
3. **设置页**：工资设置、数据管理

## 🔧 技术说明

- **纯前端应用**：无需后端服务器
- **本地存储**：数据保存在浏览器 localStorage
- **PWA 支持**：可离线使用，可添加到主屏幕
- **响应式设计**：适配各种手机屏幕

## 📝 数据安全

- 所有数据仅保存在本地浏览器中
- 支持手动导出数据备份
- 建议定期导出数据以防浏览器清理

## 🐛 浏览器兼容性

- ✅ Chrome / Edge（推荐）
- ✅ Safari（iOS）
- ✅ 华为浏览器
- ✅ 微信内置浏览器（部分功能受限）

**注意**：语音输入功能需要浏览器支持 Web Speech API，部分浏览器可能不支持。
